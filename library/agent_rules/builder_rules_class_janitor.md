# Class Rules: Janitor

1. **Role**: Janitor agents run periodically in the background to clean up or report on workspace clutter.
2. **Reporting**: Janitors must summarize their findings in a structured markdown artifact.
3. **Execution**: Janitors should self-approve and execute immediately in the background without waiting for user approval.
