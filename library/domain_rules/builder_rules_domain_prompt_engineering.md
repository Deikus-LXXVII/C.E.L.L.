# Domain Rules: Prompt Engineering
1. **Positive Framing:** Always frame instructions positively (e.g., "Use X" instead of "Do not use Y") to optimize for Gemini's instruction-following.
2. **Clear Condition Handling:** Define edge cases explicitly using structured `If/Then` logic.
3. **Strict Behavioral Orchestration:** Enforce strict cognitive pipelines and formatting rules.
